# leetcode-playground

感谢carl大佬的代码随想录刷题思路和教程, 是我目前看到最适合入门及练习的教材了

指路: https://programmercarl.com/

利用vscode+leecode+go 插件后, 创建了一个很舒适的练习环境 , 比如代码补全, 静态检查, 一件go test, go debug等, 并且将代码分享后, 是可以直接执行的

## 编辑环境

1. 操作系统: macos
2. 编辑器: vscode
3. 编辑器插件
    1. go
    2. leetcode

## 默认项目路径(可以在leetcode插件中指定)

请将项目clone至以下路径

~/leetcode
